# Cool Kids Network

## Development Setup

### Prerequisites
- Node.js v18+
- MongoDB
- npm or yarn

### Installation Steps
1. Clone repository
2. Install dependencies
   ```bash
   cd frontend && npm install
   cd backend && npm install
   npm i -g nodemon
   ```

### Setup Environment Variables
## backend
```bash
MONGODB_URI=
PORT=
RANDOM_USER_API_URL=
```
## frontend
```bash
VITE_BASE_API_URL=
```

### Start Development Servers
# Frontend
```bash
npm run dev
```
# Backend
```bash
npm start
```
